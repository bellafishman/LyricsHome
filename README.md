App Description:
Uses ReactJS frontend and NodeJS & Express backend to display and search songs to get their lyrics.

To run locally:
cd client
npm preview or npm run dev
(http://localhost:4173)
cd server
npm run dev
(http://localhost:8080)
Run both and explore site through client side
